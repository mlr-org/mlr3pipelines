# Convert an object to a Multiplicity

Convert an object to a
[`Multiplicity`](https://mlr3pipelines.mlr-org.com/dev/reference/Multiplicity.md).

## Usage

``` r
as.Multiplicity(x)
```

## Arguments

- x:

  (`any`)  
  Object to convert.

## Value

[`Multiplicity`](https://mlr3pipelines.mlr-org.com/dev/reference/Multiplicity.md)
