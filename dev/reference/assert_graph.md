# Assertion for mlr3pipelines Graph

Function that checks that a given object is a `Graph` and throws an
error if not.

## Usage

``` r
assert_graph(x)
```

## Arguments

- x:

  (`any`)  
  Object to check.

## Value

[`Graph`](https://mlr3pipelines.mlr-org.com/dev/reference/Graph.md)
`invisible(x)`
