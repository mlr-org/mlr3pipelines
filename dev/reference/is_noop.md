# Test for NO_OP

Test whether a given object is a
[`NO_OP`](https://mlr3pipelines.mlr-org.com/dev/reference/NO_OP.md).

## Usage

``` r
is_noop(x)
```

## Arguments

- x:

  `any`  
  Object to test.

## Value

`logical(1)`: Whether `x` is a `NO_OP`.
