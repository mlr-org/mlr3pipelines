# Housing Data for 506 Census Tracts of Boston

Housing Data for 506 Census Tracts of Boston

## Format

[`R6Class`](https://r6.r-lib.org/reference/R6Class.html) object
inheriting from
[`TaskRegr`](https://mlr3.mlr-org.com/reference/TaskRegr.html).

The
[`BostonHousing2`](https://rdrr.io/pkg/mlbench/man/BostonHousing.html)
dataset containing the corrected data from III AMF (1979). “The Hedonic
Price Approach to Measuring Demand for Neighborhood Characteristics.” In
*The Economics of Neighborhood*, 191–217. Elsevier.
[doi:10.1016/B978-0-12-636250-3.50015-5](https://doi.org/10.1016/B978-0-12-636250-3.50015-5)
. as provided by the `mlbench` package. See data description there.
