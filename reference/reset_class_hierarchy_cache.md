# Reset the Class Hierarchy Cache

Reset the class hierarchy cache to factory default, thereby undoing any
calls to
[`add_class_hierarchy_cache()`](https://mlr3pipelines.mlr-org.com/reference/add_class_hierarchy_cache.md)
by the user.

## Usage

``` r
reset_class_hierarchy_cache()
```

## Value

`NULL`
